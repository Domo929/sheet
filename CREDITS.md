## System Reference Document 5.2 (SRD 5.2)

This project's game data — including the spell list and spell mechanics
(levels, schools, casting times, ranges, components, durations, class lists,
saving throws, damage, and higher-level scaling) — is derived from material in
the **System Reference Document 5.2**.

### Changes made

Changes were made to the SRD 5.2 material used in this project. In particular:

- Mechanical spell data was reformatted into this application's JSON schema.
- Spell descriptions were rewritten as concise, original summaries for display
  in a terminal interface; they are not verbatim reproductions of SRD text.
